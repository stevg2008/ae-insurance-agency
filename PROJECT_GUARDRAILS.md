# A&E Insurance — Project Guardrails
## SEO Implementation Rules & Business Principles

> Reference this document before making ANY SEO, UX, content, or architectural change.

---

## Business Objective

Transform A&E Insurance into the most trusted Medicare resource in South Florida while preserving the premium A&E Insurance brand.

SEO exists to support the business. The business does NOT exist to support SEO.

Every implementation should improve:
- Trust
- Authority
- User Experience
- Accessibility
- Conversion Rate
- Local Relevance
- Long-Term Organic Growth

Never sacrifice the brand for rankings.

---

## About A&E Insurance

**Primary Service:** Medicare

**Secondary Services:** Health Insurance · Life Insurance · Retirement Planning · Critical Illness

**Primary Audience:** English-speaking adults turning 65, Medicare beneficiaries, retirees

**Primary Market:** Miami-Dade County

**Secondary Markets:** Broward County · Palm Beach County

**Long-Term Goal:** Become the most trusted Medicare authority in South Florida and one of the strongest Medicare education websites in the United States.

---

## Non-Negotiable Rules

Never:
- Keyword stuff any page
- Create doorway pages
- Create duplicate content
- Sacrifice readability
- Over-optimize title tags
- Hurt accessibility
- Create unnecessary code complexity
- Introduce technical debt
- Reduce Core Web Vitals
- Reduce conversion rates
- Weaken branding

Every page should read like it was written for a real Medicare consumer — not for a search engine.

---

## Implementation Principles

Before making ANY change, ask:

1. Would Google consider this helpful?
2. Would this improve trust?
3. Would this strengthen topical authority?
4. Would this improve local authority?
5. Would this make the website easier to use?
6. Would this strengthen E-E-A-T?
7. Would a premium digital agency approve this implementation?

If the answer is NO — choose a better solution.

---

## Execution Process

Execute ONLY ONE PHASE at a time.

**Before implementing:**
- Review `docs/implementation-plan-2026-07/qa-review.md`
- Apply every required QA modification
- Read the relevant phase prompt in full

**After implementing:**
- `npx tsc --noEmit` — zero errors required
- `npm run lint` — zero errors required
- `npm run build` — must pass

**Verify:**
- No TypeScript errors
- No lint errors
- No hydration issues
- No schema errors
- No accessibility regressions
- No broken links
- No redirect issues
- No Core Web Vital regressions
- No visual regressions

---

## After Every Phase

Produce a completion report:
- Files changed and why
- Expected SEO impact
- Expected business impact
- Risk level
- Rollback instructions
- Validation completed
- Remaining work

---

## Content Standards

- Quality over quantity
- Every article must demonstrate genuine Medicare expertise
- Every page must answer real questions real people ask
- Use clear language — written for a 64+ audience
- Avoid fluff
- Strengthen internal linking naturally
- Support every claim with accurate Medicare information from CMS/SSA/Medicare.gov

---

## Local SEO Standards

- Build genuine local authority
- Every location page must provide unique value — not a text-swap doorway page
- Strengthen Google Business Profile alignment
- Strengthen entity relationships
- Improve internal linking between service pages and location pages

---

## AI Search Standards

Optimize for: Google AI Overviews · ChatGPT · Claude · Gemini · Perplexity

Use:
- Structured information
- Strong clear headings
- Direct answers to specific questions
- FAQ content where naturally appropriate
- Strong entity relationships
- Proper schema markup

---

## Backlink Strategy Principle

Off-site authority strategy must be ethical and white-hat only.

Priority outreach targets:
- South Florida senior centers and retirement communities
- Community organizations and local nonprofits
- Local news and media
- Chambers of Commerce
- Libraries and educational resources
- Healthcare organizations
- Financial advisors, elder law attorneys, CPA firms

Never: buy backlinks · participate in link schemes · send spam outreach

---

## The Final Question

Before every implementation ask:

> **"Will this help make A&E Insurance the most trusted Medicare website in South Florida while preserving the premium brand?"**

If YES — implement it.

If NO — do not implement it.
